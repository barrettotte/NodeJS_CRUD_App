

Getting Started:
	$ heroku login
	$ git clone <repo link>
	$ cd  <cloned repo>
	$ heroku create
	Create Procfile (No extension) <Contents = web:node server.js>
	$ git push heroku master
	$ heroku ps:scale web=1
	$ heroku open

	
ClearDB MySQL AddOn:
	$ heroku addons:create cleardb:ignite
	$ heroku config | grep CLEARDB_DATABASE_URL
	Copy DB_URL
	$ heroku config:set DATABASE_URL= '<copied DB_URL>'
